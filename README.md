# CortoParalela5

## Settings

Clone the project using the following commands from console

```bash
git clone https://github.com/Marianadaso3/CortoParalela5.git
```

Run programs

Run Paralelo
```Run Paralelo
g++ paralelo.cpp -o paralelo
```

Run Secuencial
```Run Secuencial
 g++ secuencial.cpp -o secuencial 
```
